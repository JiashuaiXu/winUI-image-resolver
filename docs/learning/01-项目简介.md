# 01. 项目简介

## 什么是 Image Resolver？

Image Resolver 是一个**桌面应用程序**，用于批量提取图片的分辨率信息和 EXIF 元数据。

### 简单理解

想象你有很多照片，想要知道：
- 每张照片的尺寸（宽度和高度）
- 是用什么相机拍的
- 拍摄时间和地点
- 相机的设置（ISO、光圈等）

手动查看每张照片太麻烦了！Image Resolver 可以**自动批量处理**，一次性提取所有信息，并导出为 CSV 表格，方便分析。

## 主要功能

### 1. 📸 批量提取图片信息
- 自动扫描文件夹中的所有图片
- 提取分辨率（宽度 × 高度）
- 获取文件大小

### 2. 📋 读取 EXIF 元数据
EXIF（Exchangeable Image File Format）是嵌入在图片中的元数据，包含：
- **相机信息**: 品牌、型号
- **拍摄信息**: 日期、时间、方向
- **GPS 位置**: 经纬度坐标（如果有）
- **相机设置**: ISO、光圈、曝光时间、焦距
- **其他信息**: 软件、作者、版权等

### 3. 💾 导出为 CSV
- 将所有信息导出为 CSV 格式
- 可以用 Excel 打开分析
- 方便批量处理和统计

### 4. 🎨 现代化界面
- 美观的 WinUI 3 界面
- 支持深色/浅色主题
- 响应式布局

1. **WinUI 3**:

   - 性能优秀

2. **.NET 8.0**:
   - 跨平台支持
   - 丰富的类库
   - 强大的异步支持

3. **C#**:
   - 语法清晰易读
   - 类型安全
   - 开发效率高
