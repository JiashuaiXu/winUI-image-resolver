# 快速启动指南

## 构建和运行

### 方法 1: 使用 .NET CLI

```bash
# 还原依赖
dotnet restore

# 构建项目
dotnet build

# 运行应用
dotnet run
```

### 方法 2: 使用 Visual Studio

1. 打开 `ImageResolver.csproj`
2. 按 `F5` 运行

## 使用应用

1. **选择文件夹**: 点击顶部的 "Select Folder" 按钮，选择包含图片的文件夹（例如项目中的 `pic` 文件夹）

2. **查看图片列表**: 左侧面板会显示所有检测到的图片，包含：
   - 缩略图（加载中显示占位符图标）
   - 文件名
   - 分辨率
   - 文件大小

3. **查看详细信息**: 点击任意图片卡片，右侧面板会显示完整的 EXIF 信息，包括：
   - 基本信息（文件名、路径、分辨率、文件大小）
   - 相机信息（品牌、型号、拍摄日期）
   - 相机设置（ISO、光圈、曝光时间、焦距）
   - GPS 位置信息（如果可用）
   - 元数据（软件、作者、版权）

4. **导出 CSV**: 点击底部的 "Export to CSV" 按钮，选择保存位置，即可导出所有图片信息到 CSV 文件

## 测试数据

项目中的 `pic` 文件夹包含测试图片，可以直接使用进行测试。

## 注意事项

- 应用支持 JPEG、PNG、BMP、TIFF、GIF 格式
- 如果图片没有 EXIF 数据，相关字段会显示为 "N/A" 或留空
- 缩略图会异步加载，可能需要一些时间
- CSV 导出包含所有图片的完整信息
